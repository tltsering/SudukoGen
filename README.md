Project title: Suduko Generator
Student name: Tenzin Tsering
Short description of the project: A generated suduko game using java, made by Tenzin Tsering
How to run the program: Type in row, then column, then the number you want to insert.
Summary of how the Sudoku board is generated: A 2D array (9 x 9 grid), where you able to insert numbers by typing in the row, then column, then the actual number. The numbers in the solved puzzle is randomized 
List of files included in the project: SudokuGame.java, DesignDocument.pdf, README.md
